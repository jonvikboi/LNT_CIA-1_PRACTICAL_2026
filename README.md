# LNT CIA-1 Practical Solutions (2026)

This repository contains Node.js solution scripts for the Student Record System practical tasks.

## Directory Structure

- **Set1/**
  - `task1.js` — Welcome Message
  - `task2.js` — Declare and Update Variables
  - `task3.js` — Identify Datatypes
  - `task4.js` — Perform a Calculation
  - `task6.js` — Iterate Over Records
  - `task7.js` — Skip or Stop Early
  - `task8.js` — Add & Find
  - `task10.js` — Create and Print an Object
  - `task14.js` — Track a Date
  - `task15.js` — Apply a Math Method

## Running the Tasks

Execute any task script using Node.js from the repository root directory:

```bash
node Set1/task1.js
```
